# Vue Task Manager

Task manager application built with Vue 3, Pinia, and TypeScript.

## Features

- ✅ Task CRUD operations
- 📊 State management with Pinia
- 🧭 Vue Router integration
- 🎨 Modern Composition API
- 🧪 Vitest testing

## Installation

```bash
npm install
npm run dev
```

## Testing

```bash
npm test
```
